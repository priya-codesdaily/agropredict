# Contributing to AgroPredict

Thank you for your interest in AgroPredict!

## How to Contribute

1. Fork the repository
2. Create a feature branch
   git checkout -b feature/your-feature-name
3. Commit your changes
   git commit -m "Add: your feature description"
4. Push and create Pull Request

## Areas We Need Help

- Adding more regional languages (Odia, Telugu, Marathi)
- Improving price prediction accuracy
- Adding more states and mandi coverage
- UI/UX improvements for rural users
- Offline mode implementation

## Code Style

- Follow Flutter/Dart conventions
- Add comments for complex logic
- Test on real Android device before PR
