## 🚜 The Problem

A farmer in Odisha harvests his mango crop. He has two choices — sell today at whatever the local trader offers, or wait and risk spoilage. He has no data. He has no prediction. He has no idea what the same mango is selling for in the mandi two districts away.

**This is the daily reality of 140 million Indian farmers.**

AgroPredict gives every farmer the same market intelligence that big traders already have.

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🏪 Mandi Price Check | Enter crop + state to see live prices across mandis |
| 📊 Price Comparison Chart | Visual bar chart comparing prices across mandis |
| 🤖 Sell Now or Wait? | AI-powered advice based on price trend analysis |
| 🏆 Best Mandi Finder | Automatically finds the highest paying mandi |
| 🌾 Popular Crops | Quick access to common crops |

---

## 🛠 Tech Stack

- **Framework:** Flutter (Dart)
- **Data Source:** AGMARKNET API (Ministry of Agriculture, India)
- **Charts:** fl_chart
- **Storage:** shared_preferences
- **HTTP:** http package
